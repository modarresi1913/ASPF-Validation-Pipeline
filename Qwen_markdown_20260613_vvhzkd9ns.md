# ASPF Validation Pipeline

This repository contains the reference implementation of the Composite Character Identity Drift (cCID) metric and the LaTeX source code for the paper:  
**"Challenges and Solutions in Long-Form Realistic AI Filmmaking: A Survey and Framework Proposal" (v4.1)**.

## 📄 Paper
The full LaTeX source of the paper is available in `main.tex`. You can compile it using pdfLaTeX or XeLaTeX.
